# Portfolio Website

A personal portfolio website built with a modern frontend stack.
This site includes:

- [x] MDX and Markdown support
- [x] Optimized for SEO (sitemap, robots, JSON-LD schema)
- [x] Dynamic OG images
- [x] Syntax highlighting
- [x] Tailwind v4
- [x] daisyUI

## Demo

https://alexandrusulea.com

### Running the example

> Install dependencies and start the development server:

```bash
pnpm install
```

```bash
pnpm dev
```

The app can be accessed locally at [localhost:3000](localhost:3000)

### Linting the code

> Run the linter:

```bash
pnpm lint
```

### Testing the code

> Run the test suite:

```bash
pnpm test
```

### Makefile

Common commands are also available via the Makefile.
Run  `make` and the command.
